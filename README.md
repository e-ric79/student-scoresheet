# student-scoresheet
This is a simple Java console application that collects school details, stores student exam scores, calculates totals and averages, and displays ranked results in a formatted table.


## Purpose of the Program

This program simulates a **school grading system**.  
It takes basic school information from the user and then processes marks for 15 students across 6 subjects:

- English  
- Math  
- History  
- Geography  
- Science  
- Programming  

It outputs:

- Each studentâ€™s total marks  
- Their rank (A, B, C, D, F)  
- Subject totals  
- Subject averages  
- Overall class average  
- Number of students in each rank
